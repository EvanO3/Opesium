# Opesium 
### AI Finance Analyzer
#### A personal finance management application that uses AI to analyze spending habits and provide actionable financial insights.

## Overview

This is a **Node.js + Express backend application**  that allows users to log their expenses and receive AI-powered financial analysis. The app tracks spending across categories and uses generative AI to identify patterns, highlight wasteful spending, and suggest optimization opportunities.
Features

Expense Logging: Add expenses with amount, category, description, and date
Expense Retrieval: Fetch all logged expenses
AI Analysis: receive personalized financial feedback

Category Tracking: Organize expenses by category (Gas, Food, Entertainment, etc.)

Real-time Insights: Get actionable recommendations based on spending patterns
